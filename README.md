    计算机本科应届毕业生要有怎样的能力才算比较好的？  
    
    摘要：

    所谓认真学过专业课的程度是：

    熟练掌握数门语言
    熟练掌握一个平台
    熟练掌握、理解算法和数据结构
    掌握编译原理、操作系统、网络、图形、数据库等专业课程知识
    认可《软件工程》 

    能达到这个程度的本科生，可以随便秒大部分水硕
    
    那么回到正题，我觉得本科生具体应该学好的知识如下 ：

    语言：C和scheme，两个极端范式足够了
    数据结构和算法，《算法导论》
    计算机系统，《csapp》
    软件工程，《sicp》  

    完，这太简单了。。
